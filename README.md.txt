# Customer Personalized Offers

This is a Streamlit dashboard to analyze customer purchases and generate personalized offers.

## How to Use
1. Install dependencies: `pip install -r requirements.txt`
2. Run the Streamlit app: `streamlit run app.py`
3. Upload your own Excel or CSV dataset containing these columns:
   Invoice, StockCode, Description, Quantity, InvoiceDate, Price, Customer ID, Country
4. View RFM summary, cluster visualization, and personalized offers.
